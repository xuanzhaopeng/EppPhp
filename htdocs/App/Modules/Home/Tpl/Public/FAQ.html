Questions - Réponses


Nous espérons que vous trouverez ci-dessous les informations recherchées.


1. Qu’est-ce qu’un QR Code ?

Un QR Code est un ensemble de lettres qui sont, comme son nom l’indique, codées. Ce code se traduit sous forme de petits carrés, simples et discrets.Chaque QR Code renvoi à une adresse unique, puisqu’il correspond à une série particulière de caractères. Chaque site web ayant une adresse unique, votre page aura donc bien un qr code unique.

2. Comment « flasher » un QR Code ?

Pour « flasher » un QR Code, vous devez disposer d’un smartphone, et télécharger une application gratuite. Il existe de nombreuses applications gratuites, tapez simplement « qr code » dans votre espace de téléchargement d’applications, et prenez la première application gratuite.Toutefois vous pouvez bien entendu accéder et créer une page sans smartphone, mais directement à l’aide de votre ordinateur en vous rendant sur le site www.unsourireternel.fr.

3. Est-il possible de modifier une page créée?

Seul l’administrateur enregistré lors de la création de la page concernée peut modifier le contenu à l’aide des identifiants définis lors de la création.

4. Quels sont les paramètres de confidentialité ?

OSE vous permet de choisir lors de la création de la page entre deux types de confidentialité :

Page publique : les informations sont accessibles à tous ceux qui flashent le QR code ou recherchent la page en saisissant le nom grâce à la barre de recherche présente sur le site internet.

Page privée : la page n’est accessible qu’avec le mot de passe défini lors de la création.

6. Peut-on accéder à la page autrement qu’en flashant le QR Code ?

Vous avez bien entendu la possibilité d’accéder à la page créée en vous rendant directement sur www.unsourireternel.fr, puis en saisissant le nom recherché dans la barre de recherche du site.

7. Comment donner l'accès au profil aux proches et à la famille?

Lors de la création de la page, le QR code vous est envoyé par email afin que vous puissiez le partager avec toutes les personnes que vous voulez. Vous pouvez imprimer et apposer ce QR Code sur tous les documents que vous souhaitez (feuilles d’office religieux, Cartons, à proximité d’une photo encadrée pour que les gens puissent se rendre régulièrement et facilement sur la page..)

8. Combien de temps cette page sera-t-elle accessible?

La page créée est conçue pour une durée illimitée. Cependant, si l’administrateur souhaite la supprimer, il peut nous demander de le faire.
